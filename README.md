Finance Tracker 
A modern Android app built with Kotlin and Jetpack Compose to track income and expenses locally.


#STACK 

UI: Jetpack Compose (Material 3)

Database: Room SQLite

Networking: Retrofit & OkHttp

Images: Coil

Processing: KSP & Coroutines

#Setup 

Clone the repo.

Open in Android Studio.

Sync Gradle and Run.
